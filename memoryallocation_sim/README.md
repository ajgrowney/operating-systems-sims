> # The Buddy Allocator

## Installation
To only build the buddy allocator use:
> `$ make`

To generate this documentation in HTML use:

> `$ make doc`

To clean the project use:
> `$ make clean`

## Usage
To run the executable use:
> `$ ./buddy < test-files/test_sample1.txt`
or
> `$ ./buddy -i test-files/test_sample1.txt`
